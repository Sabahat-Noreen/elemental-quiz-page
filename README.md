# HTML & CSS MCQ Quiz Platform

A high-performance, fully responsive web-based Multiple Choice Question (MCQ) Quiz application built entirely using vanilla HTML and CSS. This platform features a dark-themed, visually engaging interface designed to test and improve knowledge of core web technologies.
